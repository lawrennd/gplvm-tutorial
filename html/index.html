Gaussian Process Tutorial - MATLAB Software

Oxford Journal Club Examples

This page will describe how to use the examples in the tutorial on the GP-LVM first presented at the Computer Vision Reading Group at Oxford's Visual Geometry group. The software is available for download here.

A tutorial paper associated with the presentation is provided here and slides used for the presentation are given here.

Some of the examples use alternative GP-LVM implementations from this site:

The GP-LVM C++ software is available from here.

The original MATLAB version of the toolbox is available here here.

The new MATLAB version of the toolbox is available here here.
